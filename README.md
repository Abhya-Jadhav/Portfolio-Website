## Installation
1. Clone the repository:
```bash
   gh repo clone Abhya-Jadhav/Portfolio-Website
```
2. Navigate to the project directory:
```bash
    cd Portfolio-Website
```
3. Open the `index.html` file in your browser to view the website.
